# 隐私与代号机制

这份文档讲清楚 `dbs-decision` 的隐私模式怎么用、什么时候开。

## 一、什么时候开

init 时 AI 问一句：**"这个决策域涉不涉及人名、商业机密、财务数据？"**

下面任一情况 → 开：

1. 关系类项目（家人、伴侣、合作方、朋友的细节）
2. 商业项目里涉及具体客户名、合作方名、价格机密
3. 财务、健康、法律相关项目
4. 用户主动说"这是私人的""不想被看到"

下面情况 → 不开：

1. 学习项目（学一门技术、读一本书）
2. 公开业务的策略思考（产品方向、内容方向）
3. 用户自己的偏好和习惯研究，不涉及他人

默认关。开关用户随时可改：`/dbs-decision 改隐私模式 on` / `off`。

## 二、开了之后多了什么

### 1. `01_事实/相关方笔记/` 文件名只用代号

```text
01_事实/相关方笔记/
├── 代号映射.md      ← 不分享、不提交
├── 相关方_01.md
├── 相关方_02.md
└── ...
```

### 2. `代号映射.md` 是唯一有真名的地方

```markdown
# 代号映射

> 这个文件不分享、不截图、不提交 git。
> 唯一作用：查代号对应的真实身份。

| 代号 | 真名 | 角色 | 加入日期 |
|---|---|---|---|
| 相关方_01 | 张三 | 主要客户 | 2026-05-01 |
| 相关方_02 | 李四 | 合作方 | 2026-05-10 |
```

### 3. AI 在对话和写文件时，默认用代号

- 用户说真名 → AI 在落盘时自动转代号
- AI 输出 → 默认用代号
- 用户问"相关方_02 是谁" → AI 查映射表，回答时只在对话里说，不再写进文件

### 4. AGENTS.md 自动写入隐私纪律段

模板：

```markdown
## 隐私纪律

1. 文件名、对话里、写文件时，默认用代号（相关方_01、相关方_02 ...）。
2. 真名只在 `01_事实/相关方笔记/代号映射.md`。
3. 这个映射文件不分享、不截图、不提交 git。
4. 用户口语化提到真名时，AI 在落盘时自动转代号，并在状态文件提示"代号映射已更新"。
5. 截图分享时，只截 `00_使用说明.md` 或脱敏文档。
```

### 5. `.gitignore` 自动追加

如果项目在 git 仓库里，init 时自动追加：

```
代号映射.md
我的当前状态.md
04_待解/
03_定格/
```

让用户能放心提交 `00_使用说明.md`、`AGENTS.md`、`02_规律/`（脱敏后）、`SOURCE_OF_TRUTH.md`。

## 三、商业项目里的隐私

商业决策里也有"敏感"的部分。例子：

- 客户名 / 合作方名 → 代号
- 具体价格 / 利润率 → 可以写"在 A 价格带"而非"¥99"
- 内部冲突 / 团队问题 → 代号 + 模糊化时间

商业项目不必把所有信息代号化。只代号化那些"如果泄漏会造成损失"的部分。

## 四、什么时候可以分享

可以分享：

- `00_使用说明.md`（脱敏的项目入口）
- `AGENTS.md`（协作规则，没有具体内容）
- `02_规律/` 里的概念和模式（已经抽象化）
- 一份脱敏过的方法论复盘

不可以分享：

- `代号映射.md`
- `我的当前状态.md`
- `01_事实/` 里的具体条目
- `03_定格/` 里的状态画像
- `04_待解/` 里的具体决策事件

## 五、降低代号心智负担的几个技巧

代号容易让对话变难读。降低负担：

### 1. 代号带角色后缀

```text
相关方_01_客户
相关方_02_合作方
相关方_03_团队成员
```

读起来"相关方_01_客户"比"相关方_01"更容易记住。

### 2. 关键代号在 `我的当前状态.md` 顶部列出别名

```markdown
> 本文件常用代号速查：
> - 相关方_01 = 主要客户
> - 相关方_03 = 重要合作方
```

用户自己看的时候一眼能对上。

### 3. 对话里 AI 可以用更口语化的代称

用户问"相关方_01 最近怎么样" → AI 答"那个主要客户最近 ..."而不是"相关方_01 最近 ..."。让代号留在文件里，对话保持自然。

## 六、隐私模式开启后的 init 流程补充

模式 A（初始化）走到第 4 步时多做这几件事：

1. 建 `01_事实/相关方笔记/代号映射.md` 空模板
2. 在 AGENTS.md 写入隐私纪律段
3. 如果项目在 git 仓库里，追加 `.gitignore`
4. 在 `00_使用说明.md` 顶部加一行"⚠️ 本项目启用隐私模式，分享前确认脱敏"

完成后告诉用户：

```text
决策项目已建立：{路径}
隐私模式：已开启
- 文件名用代号
- 真名只在 代号映射.md
- 这些文件不要提交 git：代号映射.md / 我的当前状态.md / 04_待解/ / 03_定格/
```
