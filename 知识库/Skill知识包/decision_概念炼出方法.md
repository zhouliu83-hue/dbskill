# 概念炼出方法

这份文档讲清楚 `02_规律/` 层的判断标准：什么时候一个观察可以升级成概念或模式。

## 一、不要看到一个名词就建文件

`02_规律/` 不是名词收集器。每多一个概念，未来要维护的判断就多一个。

太松：什么观察都建文件 → `02_规律/` 变成想法清单，调用不动。

太紧：等到 10 次以上才命名 → 重要模式还没被命名前就反复重演，错过窗口。

要找的是中间。

## 二、升级到 `02_规律/` 的门槛

满足下面 3 条里的 2 条，可以升级：

1. 在 3 次以上事实里出现过
2. 能解释多个 `01_事实/` 里的条目
3. 对下一步有明确指导作用

不满足的情况：

- 只有 1—2 次观察 → 先用 `[AI 暂定概念 YYYY-MM-DD]` 写在事实文件或待解文件，挂着观察
- 能命名但解释不了别的事实 → 留在 `01_事实/` 备注
- 能解释但没指导意义 → 留在 `01_事实/` 备注

## 三、命名三条标准

一个概念名字要满足三条：

### 1. 命名性

名字本身能勾起回忆，下次看到名字就想起对应模式。

好：「等外力模式」「物质前置型」「核心位 + 长期非核心位双线共存」

差：「问题 A」「关系类型 3」「现象 X」

### 2. 解释力

能套到多个事实上，让用户/AI 看完事实后说"哦原来是这个模式"。

测试方法：拿 3 条事实，用这个概念去解释，看是不是都说得通。

### 3. 工具性

能直接指导下一步动作。

例子：「等外力模式」一旦命名，下次用户又在等外力时，AI 可以直接说"你又在用这个模式"，并问"这次你愿意做发起人吗"。

如果命名后什么动作都不会改变，这个概念没工具性。

## 四、暂定标签的用法

`[AI 暂定概念 YYYY-MM-DD]` 和 `[AI 暂定模式 YYYY-MM-DD]` 是命名权限的预支。

用法：

```markdown
## 用户在重大决定前反复出现"我再想想"

[AI 暂定模式 2026-05-12] 这个反复出现的拖延行为暂时命名为「思考手淫」。
特征：决定前进入反复思考、分析、计划，但不开始。
目前观察样本：A 项目立项前 7 天 / B 合作签约前 3 天 / C 招聘决定前 2 周。
后续验证：30 天内观察是否在第 4 个决定前再次出现。
```

后续：

- 再次出现 → 在原段下追加 `[出现 YYYY-MM-DD]`
- 30 天后样本足够 → 升级成 `02_规律/模式_01_思考手淫.md`
- 30 天后没再出现 → 在原段下追加 `[未再出现 YYYY-MM-DD]`，降级标注

## 五、概念和模式怎么区分

|  | 概念 | 模式 |
|---|---|---|
| 是什么 | 一个**对象类型**或**位置类型** | 一个**行为流程**或**反复发生的序列** |
| 例子 | 「核心位」「物质前置型」「长期非核心位」 | 「等外力模式」「外驱替换」「思考手淫」 |
| 命名习惯 | 名词 | 动词或动名词 |
| 文件名 | `概念_01_*.md` | `模式_01_*.md` |

不确定就先写「概念」，后续再调整。

## 六、概念文件的版本演进

概念不是一次写死。每次新事实出现 → 在原文下追加章节，**不重写原段**。

例子：

```markdown
# 概念_01_核心位

[2026-05-09] 定义：用户认定的"长期意图、愿意为之拒绝其他关系"的对象位。
当时观察：王顺顺被列入候选，未确认。

[2026-05-15 § 6.4a] 首次有人填入：花火火。3 个月试用期。
[修正 2026-05-18] 「核心位」不等于"知道一切"——有"半透明核心位"子类。

[2026-05-25 § 7] 「核心位 + 长期非核心位」双线共存子类首次稳定 30 天。
```

读到第 3 段时，能看出概念是怎么从单点到双线、从全透明到半透明逐次精细化的。

## 七、什么时候应该把一个概念降级或废弃

满足以下任一：

1. 后续事实证伪了原定义
2. 概念被另一个更准确的概念吃掉
3. 用了一段时间发现没工具性

降级方式：

- 不删除原文件
- 在文件头部追加 `[降级 YYYY-MM-DD] 这个概念被 [概念_05_xxx] 替代，仅作为历史保留`
- 在 `02_规律/规律索引.md` 把它移到"已废弃"区

不要直接删除。降级路径本身也是数据。

## 八、规律索引文件

`02_规律/规律索引.md` 是 `02_规律/` 层的目录。每次新增/修正/降级概念都要同步。

模板：

```markdown
# 规律索引

更新时间：YYYY-MM-DD

## 概念

- [概念_01_核心位](./概念_01_核心位.md) —— 用户认定的长期意图对象位
- [概念_02_物质前置型](./概念_02_物质前置型.md) —— ...

## 模式

- [模式_01_等外力](./模式_01_等外力.md) —— 重大转折点上不当发起人
- [模式_02_外驱替换](./模式_02_外驱替换.md) —— ...

## 暂定（在 04_待解 或 01_事实 里挂着）

- [AI 暂定模式 2026-05-12]「思考手淫」—— 见 `04_待解/开放问题.md § 5`

## 已废弃

- ~~概念_03_纯损失感~~ —— [降级 2026-05-09]，被 [概念_05_交换观念] 替代
```
